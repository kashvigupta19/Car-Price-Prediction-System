# 🚗 Car Price Prediction System

## Overview

This project predicts the prices of used cars using Machine Learning techniques. The system performs data cleaning, preprocessing, exploratory data analysis (EDA), feature engineering, one-hot encoding, model training, and performance evaluation. Multiple regression models are trained and compared to identify the most effective algorithm for car price prediction.

## Features

* Data Cleaning and Preprocessing
* Missing Value Handling
* Exploratory Data Analysis (EDA)
* Feature Engineering
* One-Hot Encoding
* Train-Test Split
* Multiple Model Comparison
* Linear Regression
* Decision Tree Regressor
* Random Forest Regressor
* Gradient Boosting Regressor
* Price Prediction using the Best Performing Model

## Evaluation Metrics

* R² Score
* Mean Absolute Error (MAE)
* Root Mean Squared Error (RMSE)

## Results

Multiple regression algorithms were trained and evaluated. The best-performing model was selected based on R² Score, MAE, and RMSE, providing accurate car price predictions on unseen data.
